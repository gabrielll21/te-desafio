# Site de Desafios entre Amigos

## Descrição
Este projeto é um site onde amigos podem criar e participar de desafios. O objetivo é promover a interação e competição saudável entre amigos.

## Funcionalidades
- Criação de desafios personalizados
- Participação em desafios existentes
- Sistema de pontuação e ranking
- Comentários e feedbacks nos desafios

## Instalação
1. Clone o repositório: `git clone https://github.com/usuario/site-desafios.git`
2. Navegue até o diretório do projeto: `cd site-desafios`
3. Instale as dependências: `npm install`

## Uso
1. Inicie o servidor de desenvolvimento: `npm start`
2. Acesse o site em `http://localhost:3000`
